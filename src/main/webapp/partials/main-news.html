Welcome to EveKit!

EveKit is a web-based toolkit for building applications based on the EVE Online API and EVE Online static data export. You can configure EveKit to automatically download your Character and Corporation data, then use REST or one of the supported client APIs (currently: Java, Javascript and PHP) to access your data.

Key features:

EveKit can download for both characters and corporations, at least as frequently as the EVE API servers allow.
EveKit synchronizes your data on each download and keeps a complete history of everything. Want to know what your assets looked like a month ago? No problem. The EveKit APIs allow you to access your data at a date you specify (defaulting to the latest live data).
EveKit lets you attach persistent meta data to your synchronized data. This lets you do things like tag your blueprint originals with the appropriate ME and PE (not really needed as of Hyperion, yay!).
The EveKit access API supports a key and hash scheme, similar to the EVE Online API key scheme, so you can give different people different access to your data.
Your synchronized data is always available in bulk form. We recognize that not everyone will want to use the EveKit APIs to access their data. You can request a snapshot of your data directly from the accounts page at any time. We'll send you the snapshot in CSV format. We don't trap your data behind our APIs, you can get a complete snapshot any time you like. See the Account FAQ for details on how often snapshots are created.

Getting Started:

1 Sign in using one of the third party providers listed in the upper right corner. At EveKit, we don't maintain login credentials (who wants that headache?). We only pull basic identification via OAuth from the third party you use for sign in.
2 Once you're logged in, select "Sync Accounts -> Create Sync Account" from the menu.
3 On the "Add Account" page, you'll need to:
  • Provide an EVE Online API key and vCode to use for synchronization.
  • Select whether to synchronize a character or corporation.
  • Select which character to use for synchronization.
  • Select whether you want to synchronize automatically or manually.
4 That's it! To access your data online, create a data access key by selecting "Data Access Keys -> Data Access Keys for <your_account>". If you prefer offline access, you can click "Download Latest Snapshot" from the account list (we'll download your data as a zip of CSV files). You can find more information about accounts and access keys on the Account FAQ.

About EveKit

  • EveKit is provided by Dark Planet Ventures (DPVT), a one player - multiple alts extremely tiny EVE corporation.
  • Federated login decals from CSS3 Social Signin Buttons.
  • Static EVE data is converted from CCP's export using Desmont McCallock's EVE SDE to SQL tool.
