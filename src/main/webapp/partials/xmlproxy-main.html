EVE Online XML Endpoints Proxy

The EveKit site can be used to proxy connections to the EVE Online XML endpoints. Some reasons you might do this are:
  • You want to process JSON output (which is what this proxy returns) instead of XML output;
  • You lack an XML API library in your language of choice; or,
  • You want someone else to worry about staying up to date with changes in the XML endpoints.
The EveKit proxy is nothing more than a live version of the Orbital EVE XML API Proxy GitHub project. The REST calls provided by the proxy are annotated via Swagger which makes it very easy to generate proxy clients in a variety of languages. The GitHub project page contains instructions for generating proxy clients, or even running your own proxy (if you so desire). The Swagger definition for the live proxy can be found here.

You can experiment with the proxy using the Swagger UI panel.
